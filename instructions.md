Prerequisites
-------------

* virtualenv
* virtualenvwrapper
* nodejs / npm
* python-pip


Setup
-----

** Install core JS dependencies **

install Browserify:
```
$ sudo npm install -g browserify
```

also install uglify-js:
```
$ sudo npm install -g uglify-js
```

lastly, install Babel plugin for Browserify
```
$ sudo npm install --save-dev babelify
```
